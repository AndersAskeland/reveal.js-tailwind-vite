# reveal.js-tailwind-vite
Template for creating reveal.js presentations with tailwind styling and Vite dev server and build commands. Uses viteSingleFile to build to single HTML.

## How to use
**Clone repository**
```
git clone 
```

**Install dependencies**
```
npm install
```

**Run dev server**
```
npm run dev
```

**Build**
```
npm run build
```
